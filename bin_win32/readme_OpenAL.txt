AssaultCube uses a customized version of OpenAL-Soft 1.16.0

AssaultCube can use deprecated, but official OpenAL redistributable provided
by Creative Labs. If you encounter any sound problems you can switch to official OpenAL,
this can be done by renaming OpenAL32.dll to OpenAL32_RemoveThisPartToUseOpenAL-Soft.dll.
You must have installed official OpenAL in the system, if you don't have, please run
oalinst.exe installer from http://openal.org/ .
